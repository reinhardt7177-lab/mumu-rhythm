# MUMU 리듬 스튜디오

빠른 클래식 네 곡을 실제 악기 기반 편곡으로 들으며, 위에서 떨어지는 노트 바를 직접 터뜨려 가락, 반복, 강박, 대비를 발견하는 초등학생용 2D 음악 감상 활동입니다.

## 실행

```powershell
npm install
npm run dev
```

기본 주소는 `http://127.0.0.1:5173`입니다.

## 빌드

```powershell
npm run build
```

## 조작

- PC: 떨어지는 노트 바를 마우스로 클릭
- 태블릿·모바일: 떨어지는 노트 바를 직접 터치
- 일시정지: `Escape`

판정선에 입력 시간을 맞추는 방식이 아닙니다. 화면에 보이는 동안 바를 누르면 해당 음이 나고 그 자리에서 터집니다. 결과에는 타이밍 정확도 대신 노트 수집률과 연속 팝이 기록됩니다.

## 수록곡

- 모차르트: 터키 행진곡
- 오펜바흐: 캉캉
- 로시니: 윌리엄 텔 서곡 피날레
- 브람스: 헝가리 무곡 5번

모든 곡은 54~72초의 수업용 편곡이며, 연주를 쉬고 오케스트라 층을 듣는 집중 감상 구간을 포함합니다.

## 음악 제작

악보 타임라인, 반주, 화면 노트를 하나의 빌드 스크립트에서 생성합니다. 최초 실행 시 빌드 도구와 SoundFont를 `tmp/`에 내려받으며, 저장소에는 완성된 OGG만 포함합니다.

```powershell
pip install music21
python tools/build_music.py
```

- Cadenza의 MIT 공개 선율 데이터를 바탕으로 게임용 편곡을 제작합니다.
- FluidSynth와 GeneralUser GS SoundFont로 반주를 오프라인 렌더링합니다.
- 음별 히트 사운드는 피치 변경 없이 37개 반음 샘플을 담은 악기별 오디오 스프라이트로 생성합니다.
- 브라우저는 선택한 곡의 반주와 히트 스프라이트만 읽습니다.

출처와 라이선스는 `THIRD_PARTY_ASSETS.md`에 정리되어 있습니다.
