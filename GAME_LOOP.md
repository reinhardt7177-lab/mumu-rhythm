# Game Loop

## Stage Loop

1. Setup: 곡 카드와 한 가지 감상 미션을 본다.
2. Action: 위에서 떨어지는 가로 노트 바를 마우스나 손가락으로 직접 누른다.
3. Check: 보이는 노트 바의 실제 2D 영역과 입력 좌표가 만났는지 확인한다.
4. Feedback: 누른 위치에서 색 입자와 음표가 터지고 곡과 같은 실제 악기음이 난다.
5. Progress: 곡을 끝까지 듣고 노트 수집 별과 감상 배지를 받는다.

## Student Actions

- 곡 선택
- 미리 듣기
- 떨어지는 노트 바 직접 클릭·터치
- 감상 질문 선택
- 재연주 또는 다른 곡 이동

## Feedback and Hints

- 빈 공간 입력: 아무 감점 없이 다음 노트 탐색
- 놓친 노트: 별도 경고 없이 화면 아래로 자연스럽게 이동
- 감상 질문 오답: 한 가지 소리 단서를 제시하고 바로 다시 선택
- 곡은 어떤 결과에서도 중단하지 않는다.

## Reward System

- 수집 별: 전체 노트 중 터뜨린 비율 기준 1~3개
- 감상 배지: 곡별 질문을 듣고 해결하면 획득
- 최고 기록: 기기 로컬에 곡별 저장

## Accessibility and Classroom Notes

- 색 외에 낮은음은 왼쪽, 높은음은 오른쪽이라는 위치 규칙으로 구분
- 실제 렌더링된 노트 바 좌표에 추가 터치 여백 적용
- 동작 감소 설정 지원
- 교사는 설정에서 전체 음량 조절과 기록 초기화 가능
