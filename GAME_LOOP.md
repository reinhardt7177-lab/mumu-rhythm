# Game Loop

## Stage Loop

1. Setup: 곡 카드와 한 가지 감상 미션을 본다.
2. Action: 실제 가락의 높낮이에 따라 5레인 노트를 누르고 긴 음은 유지한다.
3. Check: Web Audio 시계 기준으로 입력 시점을 판정한다.
4. Feedback: 판정 색, 곡과 같은 실제 악기음, 레인 빛으로 즉시 반응한다.
5. Progress: 곡을 끝까지 듣고 리듬 배지와 감상 배지를 받는다.

## Student Actions

- 곡 선택
- 미리 듣기
- A/S/D/J/K 또는 화면 패드 연주
- 감상 질문 선택
- 재연주 또는 다른 곡 이동

## Feedback and Hints

- 너무 이른 입력: "조금만 기다려요"
- 연속 MISS: 판정 범위를 잠시 시각적으로 강조
- 감상 질문 오답: 한 가지 소리 단서를 제시하고 바로 다시 선택
- 곡은 어떤 결과에서도 중단하지 않는다.

## Reward System

- 리듬 별: 타이밍 정확도 기준 1~3개
- 감상 배지: 곡별 질문을 듣고 해결하면 획득
- 최고 기록: 기기 로컬에 곡별 저장

## Accessibility and Classroom Notes

- 색 외에 레인 위치와 키 문자로 구분
- 노트와 터치 패드는 동일한 실측 2D 좌표를 사용
- 동작 감소 설정 지원
- 효과음, 반주, 전체 음량 분리 예정
- 교사는 설정에서 판정 보정과 기록 초기화 가능
