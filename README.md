# MUMU 리듬 스튜디오

빠른 클래식 네 곡의 실제 연주를 들으며, 위에서 떨어지는 노트 바를 직접 터뜨려 가락, 반복, 강박, 대비를 발견하는 초등학생용 2D 음악 감상 활동입니다.

## 실행

```powershell
npm install
npm run dev
```

기본 주소는 `http://127.0.0.1:5173`입니다.

## 빌드

```powershell
npm run build
```

## 조작

- PC: 짧은 노트 바는 클릭하고 긴 노트 바는 누른 채 유지
- 태블릿·모바일: 짧은 노트 바는 터치하고 긴 노트 바는 손가락으로 길게 누르기
- 일시정지: `Escape`

판정선에 입력 시간을 맞추는 방식이 아닙니다. 짧은 바는 한 번 누르면 터지고, 긴 바는 밝은 진행 표시가 끝까지 찰 때까지 누르면 터집니다. 결과에는 타이밍 정확도 대신 노트 수집률과 연속 팝이 기록됩니다.

## 수록곡

- 모차르트: 터키 행진곡
- 오펜바흐: 캉캉
- 로시니: 윌리엄 텔 서곡 피날레
- 브람스: 헝가리 무곡 5번

모든 곡은 70~76초의 실제 연주 발췌본이며, 녹음의 빠르기와 프레이즈를 따라 만든 전용 노트 차트를 사용합니다.

## 음악 제작

공개 라이선스의 실제 연주를 수업용 길이로 편집하고, 녹음의 어택과 음역을 분석해 화면 노트를 생성합니다. 원본은 `tmp/`에 내려받으며 저장소에는 편집한 OGG만 포함합니다.

```powershell
pip install librosa
python tools/build_real_music.py
```

- FFmpeg로 구간 편집, 페이드, 라우드니스 정규화를 처리합니다.
- librosa의 온셋 강도와 주된 음역을 이용해 곡별 5레인 차트를 만듭니다.
- 노트 팝은 음정이 없는 짧은 Web Audio 효과라 원연주의 화성과 충돌하지 않습니다.
- 브라우저는 선택한 곡의 실제 연주 OGG만 먼저 읽습니다.

출처와 라이선스는 `THIRD_PARTY_ASSETS.md`에 정리되어 있습니다.
